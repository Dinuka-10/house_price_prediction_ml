🏠 House Price Prediction - Machine Learning Project

Overview

This project focuses on predicting house prices using Machine Learning techniques. It estimates property prices based on features such as location, area, number of bedrooms, and other relevant attributes.

⚠️ Note: This repository contains only the ML part. Trained model pickle files are not included.

Features

Exploratory Data Analysis (EDA) on the dataset.

Data preprocessing

Model Training using algorithms like:

Random Forest Regressor
XGBoost Regressor

Model Evaluation using:

Mean Absolute Error (MAE), 
Mean Squared Error (MSE), 
Root Mean Squared Error (RMSE), 
R² Score

Visualizations for feature importance and performance comparison.
